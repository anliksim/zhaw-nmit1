Anhang zu Aufgabe 3
--------------------

Es gibt 3 Scripts zu Aufgabe 3:

1. Anliker_Simon_IT16ta_ZH09_S3_Aufg3.m
2. Anliker_Simon_IT16ta_ZH09_S3_Aufg3_w_vec.m
3. Anliker_Simon_IT16ta_ZH09_S3_Aufg3_performance.m

Das erste File ist die Weiterführung der Vorgaben/Hinweise im Praktikum.
Nach der Fertigstellung haben wir uns gefragt, ob es nicht eine bessere
Lösung unter der Verwendung von Linienvektoren geben würde. Das zweite
File speichert die umgewandelten Zahlen zuerst als Zahl in einen Vektor.
Das dritte File vergleicht die Performance der beiden Varianten wobei
die Variante mit Vektoren grundsätzlich besser abschneidet.
